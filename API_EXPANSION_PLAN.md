# План расширения API функционала для HopSpot

## Текущая архитектура проекта

**HopSpot** - это iOS приложение с архитектурой MVP, которое:
- Загружает список пива из Punk API
- Отображает пиво в табличном виде
- Имеет функцию избранного с локальной базой данных Realm
- Использует стандартные компоненты UIKit

## Предложения по расширению API функционала

### 1. Расширение модели данных Beer
```swift
struct Beer: Codable {
    var id: Int?
    var name: String
    var tagline: String
    var description: String?
    var abv: Double?
    var ibu: Double?
    var ebc: Double?
    var srm: Double?
    var ph: Double?
    var food_pairing: [String]?
    var brewers_tips: String?
    var image_url: String?
    var ingredients: Ingredients?
    var method: Method?
}
```

### 2. Новые API эндпоинты в NetworkManager

- **Поиск по названию**: `getBeerByName(name: String)`
- **Фильтрация по ABV**: `getBeersByABV(min: Double, max: Double)`
- **Фильтрация по IBU**: `getBeersByIBU(min: Double, max: Double)`
- **Случайное пиво**: `getRandomBeer()`
- **Пагинация**: `getBeers(page: Int, perPage: Int)`
- **Поиск по ингредиентам**: `getBeersByIngredient(ingredient: String)`
